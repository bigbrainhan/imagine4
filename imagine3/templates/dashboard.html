Outputs and Use

Patient Progression: The chatbot will act as a therapist, and update you with any significant progressions. Whether this means more physical meetings, or medicine administration, the use of the chatbot is to give you insights of the patient.

Media Upload and Diagnosis: Our model takes in a .wav file (audio), a .mat file (eeg), and text input to diagnose the patient, and create the case-specific chatbot. Our model is very accurate, but is only used for initial setup and diagnostics (it can be used for intermediate checkups if needed).
